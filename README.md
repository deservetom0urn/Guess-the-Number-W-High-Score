# Python Guessing Game with High Score

A CLI-based number guessing game (1–20) featuring a high score tracking system using file I/O in Python.

### ✨ Key Features
* **File Handling (I/O):** Reads and permanently updates the high score via the `Datas.txt` file.
* **Smart Feedback:** Provides “too big” or “too small” hints after each guess.
* **Game Loop & Logic:** Uses a `while loop` to count the player’s attempts in *real-time*.

### 📋 How to Run
1. Create a `Datas.txt` file in the same folder, then enter a number on the second line (e.g., `99`).
2. Run the command: `python main.py`
