# Python Guessing Game with High Score

This is a simple command-line number guessing game built with Python. The player tries to guess a random number between 1 and 20, while the game keeps track of the best score using a text file.

🛠️ What I Built
Generates a random number for the player to guess.
Gives hints if the guess is too high or too low.
Counts the number of attempts needed to guess the correct number.
Saves the best score so it can be used the next time the game is played.
📋 Features
Built with Python
Uses the random library to generate a random number
Uses file handling to save and update the high score in Datas.txt
Provides feedback after every guess
Uses loops and conditional statements to control the game flow
▶️ How to Run
Create a file named Datas.txt in the same folder as the project.
Add a number on the second line (for example, 99) as the initial high score.
Run the program using:
python main.py
